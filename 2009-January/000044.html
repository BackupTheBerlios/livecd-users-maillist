Hey, great to hear that you finally got the ISO. :)

Yeah, that script, for obvious reasons, uses the PCLOS gfxboot theme
by default. But you can change that by simply using the "-boottheme"
modifier with the command you use to create the ISO.

Issue mklivecd --help to learn more about it.
